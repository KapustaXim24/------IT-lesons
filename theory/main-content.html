На головну

1. Що таке HTML?

HTML (HyperText Markup Language) — це мова розмітки, яка використовується для створення структури веб-сторінок. Уявіть, що ви будуєте будинок. HTML — це його каркас: стіни, дах, поверхи. Він визначає, де буде заголовок, де текст, де зображення, але не каже, якого вони будуть кольору чи розміру.

Основні елементи (теги):

✅

...

— найбільший заголовок.

✅

...

— абзац тексту.

✅ — тег для вставки зображення.

✅ ... — тег для створення посилання.

Приклад структури HTML-документа:



  
    Моя перша сторінка
  
  
    
Привіт, світ!

    
Це моя перша веб-сторінка.

2. Що таке CSS?

CSS (Cascading Style Sheets) — це мова стилів, яка відповідає за зовнішній вигляд веб-сторінки. Якщо HTML — це каркас будинку, то CSS — це його дизайн: колір стін, шпалери, меблі, картини. CSS дозволяє нам зробити сторінку красивою та унікальною.

Як працює CSS?

CSS працює за принципом "селектор {властивість: значення;}".

➡️ Селектор — це те, до якого HTML-елемента ми застосовуємо стиль (наприклад, до всіх абзаців p).

➡️ Властивість — що саме ми хочемо змінити (наприклад, color — колір тексту).

➡️ Значення — як ми хочемо це змінити (наприклад, blue — синій).

Приклад CSS-коду:

p {
    color: blue; /* Робить текст всіх абзаців синім */
    font-size: 18px; /* Встановлює розмір шрифту 18 пікселів */
}

Розроблено в рамках лабораторної роботи. 2024
